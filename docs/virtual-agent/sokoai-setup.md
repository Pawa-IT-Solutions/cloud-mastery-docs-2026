# Phase 1: Initial Setup

In this phase, you will log into the CX Agent Studio, select your project, and create the SokoAI agent application.

---

## Step 1: Access Vertex AI Agent Builder

1. Log in to the [CX Agent Studio console](https://ces.cloud.google.com) using your training GCP account from Session 1.

2. You will see a page like this:

    ![CX Agent Studio landing page](assets/<!-- ADD IMAGE: image1 equivalent - CX Agent Studio landing page -->)

3. **Select your project.** Click the project dropdown at the top and select **View all projects**.

    ![View all projects dropdown](assets/<!-- ADD IMAGE: image2 equivalent - View all projects dropdown -->)

4. Find and select your assigned project (e.g. *sean mwiti*), then click **Continue**.

    ![Project selected and confirmed](assets/<!-- ADD IMAGE: image3 equivalent - Project selected confirmed -->)

---

## Step 2: Create the AI App

1. In the middle of the screen, locate the **Create your first AI agent** section and click on it.

    !!! note
        Ensure the region defaults to **US (Multiple regions)** before proceeding.

    ![Create your first AI agent](assets/<!-- ADD IMAGE: image4 equivalent - Create AI agent section -->)

    ![Region selection - US Multiple regions](assets/<!-- ADD IMAGE: image5 equivalent - Region US selection -->)

2. Enter a name for your agent — use `sokoai-agent` — then press **Create**.

    ![Enter agent name sokoai-agent](assets/<!-- ADD IMAGE: image6 equivalent - Agent name input -->)

3. Once the agent is created, click on it to select it.

    ![Select the newly created agent](assets/<!-- ADD IMAGE: image7 equivalent - Select agent -->)

4. This opens the **Agent Builder** page where you will configure all components.

    ![Agent Builder main page](assets/<!-- ADD IMAGE: image8 equivalent - Agent Builder main page -->)

---

!!! success "Phase 1 Complete"
    Your SokoAI agent application is now created. In the next phase, you will build the root playbook and the four specialist sub-agents.
